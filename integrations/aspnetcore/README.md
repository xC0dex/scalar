# Scalar .NET Integrations

[![Discord](https://img.shields.io/discord/1135330207960678410?style=flat&color=5865F2)](https://discord.gg/scalar)

This repository contains .NET integrations that provide easy ways to render beautiful API references based on OpenAPI/Swagger documents.

## Available Integrations

### Scalar.AspNetCore

[![Version](https://img.shields.io/nuget/v/Scalar.AspNetCore)](https://www.nuget.org/packages/Scalar.AspNetCore)
[![Downloads](https://img.shields.io/nuget/dt/Scalar.AspNetCore)](https://www.nuget.org/packages/Scalar.AspNetCore)

The `Scalar.AspNetCore` package provides integration for ASP.NET Core applications. It enables you to easily add Scalar API reference documentation to your web applications.

[Learn more about Scalar.AspNetCore](./aspnetcore/README.md)

### Scalar.Aspire

[![Version](https://img.shields.io/nuget/v/Scalar.Aspire)](https://www.nuget.org/packages/Scalar.Aspire)
[![Downloads](https://img.shields.io/nuget/dt/Scalar.Aspire)](https://www.nuget.org/packages/Scalar.Aspire)

The `Scalar.Aspire` package provides integration for .NET Aspire applications. It enables you to easily add Scalar API reference documentation to your Aspire-powered applications.

[Learn more about Scalar.Aspire](./aspire/README.md)

## Features

- Stunning API Reference
- Compatible with .NET 8 and above
- Independent of OpenAPI document generators
- Fully AOT (Ahead-of-Time) compatible
